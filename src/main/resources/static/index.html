API Documentation

About the Reach API

The KB Query Server (API) provides a REST-ful interface to biological grounding information based the CLU Lab Bioresources data. The API services accept REST calls (via an HTTP GET) or JSON calls (via an HTTP POST). The results are returned in JSON format.

Send an NXML File

Send an NXML file for processing via an HTTP POST, receive results in either FRIES or IndexCard JSON format.
${appContext}/${controllerName}/nxml
POST
nxml
[ output=fries | output=indexcard ]
curl -XPOST -F 'nxml=<PMC1240239.nxml' 'http://agathon.sista.arizona.edu:8080${appContext}/${controllerName}/nxml'
curl -XPOST -F 'nxml=<PMC1240239.nxml' 'http://agathon.sista.arizona.edu:8080${appContext}/${controllerName}/nxml?output=indexcard'

Send Plain Text

Send a plain text snippet for processing via an HTTP POST, receive results in either FRIES or IndexCard JSON format.
${appContext}/${controllerName}/text
POST
text
[ output=fries | output=indexcard ]
curl -XPOST 'http://agathon.sista.arizona.edu:8080${appContext}/${controllerName}/text' -d 'text=A Sos-1-E3b1 complex directs Rac activation by entering into a tricomplex with Eps8.'
curl -XPOST 'http://agathon.sista.arizona.edu:8080${appContext}/${controllerName}/text' -d 'text=A Sos-1-E3b1 complex directs Rac activation by entering into a tricomplex with Eps8.' -d 'output=indexcard'

NXML Uploader

Upload an NXML file for processing from your local machine through a browser form. The processed results are returned to your browser in FRIES or IndexCard JSON format.
