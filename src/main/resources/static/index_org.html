processors-server

Web interface to the CLU Lab's processors NLP library

Services

Endpoint Request type Request form Description
/annotate POST {"text":""} Produces a Document using FastNLPProcessor
/fastnlp/annotate POST {"text":""} Produces a Document using FastNLPProcessor
/bionlp/annotate POST {"text":""} Produces a Document using BioNLPProcessor
/sentiment/corenlp/score POST Document, Sentence, or {"text": ""} Produces a sentiment score for each sentence using FastNLPProcessor. Responses are of the form {"scores": []}
/odin/extract POST Recieves a POST of one of the following forms:
  1. {"rules": "", "text": ""}
  2. {"rules": "", "document": {}}
  3. {"url": "", "text": ""}
  4. {"url": "", "document": {}}
a "url" value should end in .yml. 		Responses are of the form {"mentions": []}
/shutdown POST Any POST to this endpoint will shutdown the server.
